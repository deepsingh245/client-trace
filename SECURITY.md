# Security Policy

## Reporting a Vulnerability

We take the security of `client-trace` seriously. If you have discovered a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them privately by emailing deepsingh.245@gmail.com.

We will acknowledge your report within 48 hours and will do our best to triage and fix the vulnerability as soon as possible. We will try to keep you informed about our progress throughout the process.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |
